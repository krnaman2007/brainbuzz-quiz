# 🧠 BrainBuzz - Quiz App
A fun and interactive quiz app built with HTML, CSS & JavaScript.

## ✨ Features
- 3 difficulty levels — Easy, Medium, Hard
- 15 second timer per question with danger mode
- Shuffle questions every time
- Instant answer feedback — green/red highlight
- Progress bar with smooth animation
- Performance review after quiz
- Confetti animation on full score 🎉
- Total time tracker

## 🛠️ Built With
- HTML
- CSS
- JavaScript
